# เว็บแอพคำนวณภาษีเงินได้บุคคลธรรมดา ปี 2568 (ยื่นในปี 2569)

เว็บแอพพลิเคชันสำหรับคำนวณภาษีเงินได้บุคคลธรรมดาปี 2568 (ยื่นในปี 2569) พัฒนาด้วย Python และ Streamlit

## ฟีเจอร์

- คำนวณภาษีเงินได้บุคคลธรรมดาตามอัตราภาษีปี 2568 (ยื่นในปี 2569)
- รองรับค่าลดหย่อนครบถ้วน (ส่วนตัว, ครอบครัว, ประกัน, การลงทุน, เงินบริจาค)
- บันทึกผลการคำนวณเป็นรายบุคคล
- ดูประวัติการคำนวณ
- UI สวยงามและใช้งานง่าย รองรับมือถือและแท็บเล็ต

## การติดตั้ง

1. ติดตั้ง dependencies:
```bash
pip install -r requirements.txt
```

2. รันแอพ:
```bash
streamlit run app.py
```

## โครงสร้างโปรเจกต์

- `app.py` - ไฟล์หลัก Streamlit
- `tax_calculator.py` - ฟังก์ชันคำนวณภาษี
- `database.py` - จัดการฐานข้อมูล SQLite
- `tax.db` - ฐานข้อมูล (สร้างอัตโนมัติ)

## อัตราภาษีปี 2568 (ยื่นในปี 2569)

### ระยะเวลายื่นภาษี
- **ยื่นแบบกระดาษ**: 1 มกราคม - 31 มีนาคม 2569
- **ยื่นแบบออนไลน์**: 1 มกราคม - 8 เมษายน 2569

### อัตราภาษีขั้นบันได

- 0 - 150,000 บาท: ยกเว้นภาษี
- 150,001 - 300,000 บาท: 5%
- 300,001 - 500,000 บาท: 10%
- 500,001 - 750,000 บาท: 15%
- 750,001 - 1,000,000 บาท: 20%
- 1,000,001 - 2,000,000 บาท: 25%
- 2,000,001 - 5,000,000 บาท: 30%
- 5,000,001 บาทขึ้นไป: 35%

## ค่าลดหย่อนที่รองรับ

### ส่วนตัวและครอบครัว
- ค่าลดหย่อนส่วนตัว: 60,000 บาท
- คู่สมรส: 60,000 บาท
- บุตร: 30,000 บาท/คน (บุตรคนที่ 2 เกิดหลังปี 2561: 60,000 บาท)
- ค่าเลี้ยงดูบิดามารดา: 30,000 บาท/คน

### ประกัน
- เบี้ยประกันชีวิต: สูงสุด 100,000 บาท
- เบี้ยประกันสุขภาพตนเอง: สูงสุด 25,000 บาท
- เบี้ยประกันสุขภาพบิดามารดา: สูงสุด 15,000 บาท

### การลงทุน
- กองทุน RMF/SSF: สูงสุด 500,000 บาท
- กองทุน Thai ESG: สูงสุด 300,000 บาท
- กองทุนสำรองเลี้ยงชีพ (PVD): สูงสุด 15% ของเงินได้ แต่ไม่เกิน 500,000 บาท

### อื่นๆ
- Easy E-Receipt: 50,000 บาท
- ดอกเบี้ยที่อยู่อาศัย: สูงสุด 100,000 บาท
- เงินบริจาคทั่วไป: สูงสุด 10% ของเงินได้หลังหักค่าลดหย่อน
- เงินบริจาคเพื่อการศึกษา: 2 เท่าของเงินบริจาค แต่ไม่เกิน 10% ของเงินได้หลังหักค่าลดหย่อน

## หมายเหตุสำคัญ

**ปีภาษี 2568** หมายถึง รายได้ที่ได้รับระหว่างวันที่ **1 มกราคม - 31 ธันวาคม 2568**

**ปีที่ต้องยื่นภาษี 2569** หมายถึง ต้องยื่นแบบแสดงรายการภาษีและชำระภาษีในปี **2569** ตามระยะเวลาที่กำหนดข้างต้น

การคำนวณภาษีในระบบนี้ได้รับการตรวจสอบความถูกต้องตามมาตรฐานภาษีเงินได้บุคคลธรรมดาปี 2568 ของกรมสรรพากร
